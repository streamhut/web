# streamhut-web

> Web app for [streamhut](https://github.com/streamhut/streamhut)

[![License](http://img.shields.io/badge/license-MIT-blue.svg)](https://raw.githubusercontent.com/streamhut/web/master/LICENSE)

## Development

Install dependencies:

```bash
npm install
```

Start application:

```bash
npm run dev
```

Build:

```bash
npm run build
```
